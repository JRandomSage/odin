## Summary
Kill a job with the specified ID - not all jobs are killable though. Killable jobs are `keylog`, `triagedirectory`, and `portscan`.


- Needs Admin: False  
- Version: 1  
- Author: @xorrior

## Usage

```
jobkill SOME-GUID-GOES-HERE
```


## Detailed Summary

Kill a running job
